# GitDigital Grant Structure

**Version 2.0 | Last Updated: March 2026**

## Overview

The GitDigital Grants Program funds development, research, and community initiatives that advance the Aurora ZK Cryptography Framework and the broader GitDigital ecosystem. We believe in supporting contributors who share our vision for transparent, secure, and accessible cryptography.

## Grant Philosophy

- **Open to all** – Anyone can apply regardless of location or background
- **Merit-based** – Funding based on quality, impact, and feasibility
- **Transparent** – All applications, decisions, and outcomes are public
- **Milestone-driven** – Funds released upon verifiable completion
- **Community-governed** – Decisions made by Technical Advisory Board

## Grant Tracks

### Track 1: Core Development

Funding for direct contributions to the Aurora framework.

| Type | Description | Range | Duration |
|------|-------------|-------|----------|
| Feature Development | New cryptographic primitives, protocols | $5k-$50k | 1-6 months |
| Performance Optimization | Speed, memory, or size improvements | $2k-$25k | 1-3 months |
| Security Hardening | Audit preparation, fuzzing, formal verification | $5k-$40k | 1-4 months |
| Infrastructure | CI/CD, testing, deployment tooling | $2k-$20k | 1-2 months |

### Track 2: Research

Funding for exploratory work and novel contributions.

| Type | Description | Range | Duration |
|------|-------------|-------|----------|
| Theoretical Research | New ZK constructions, proofs | $10k-$75k | 3-12 months |
| Applied Research | Implementation studies, benchmarks | $5k-$30k | 2-6 months |
| Literature Review | Comprehensive analysis of specific areas | $2k-$10k | 1-3 months |

### Track 3: Developer Ecosystem

Funding for tools and resources that help developers use Aurora.

| Type | Description | Range | Duration |
|------|-------------|-------|----------|
| Documentation | Tutorials, guides, API references | $2k-$20k | 1-4 months |
| Developer Tools | SDKs, CLIs, debugging tools | $5k-$30k | 1-4 months |
| Example Projects | Demonstrations, templates, starter kits | $1k-$10k | 1-2 months |

### Track 4: Community

Funding for events, education, and community building.

| Type | Description | Range | Duration |
|------|-------------|-------|----------|
| Events | Hackathons, workshops, meetups | $1k-$20k | 1-time |
| Educational Content | Videos, courses, tutorials | $1k-$15k | 1-3 months |
| Translations | Documentation localization | $500-$5k | 1-2 months |

## Grant Tiers

| Tier | Amount | Review | Voting | Reporting |
|------|--------|--------|--------|-----------|
| Micro | $500 - $2,000 | Fast-track | TAB Chair | Simple |
| Small | $2,001 - $10,000 | TAB review | TAB consensus | Standard |
| Medium | $10,001 - $50,000 | TAB + external review | TAB vote | Detailed |
| Large | $50,001+ | Full due diligence | Core + Community | Comprehensive |
