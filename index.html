Solana KYC Compliance SDK

Solana KYC Compliance SDK

A robust developer toolkit to integrate seamless Know Your Customer (KYC) and Anti-Money Laundering (AML) checks directly into your Solana applications. Ensure regulatory compliance with minimal friction.

Get Started View on GitHub

Overview

This SDK simplifies the complex process of embedding regulatory compliance into decentralized applications (dApps) on the Solana blockchain. It provides a unified interface for identity verification, risk assessment, and compliance reporting, allowing you to focus on building your core product.

Core Features

Identity Verification

Integrate with leading providers for document checks, liveness detection, and biometric verification. Store results as on-chain attestations.

Risk Scoring

Automated AML screening and transaction monitoring. Real-time risk scoring for wallets and transactions based on global watchlists.

Program Library

A set of on-chain Solana programs for managing verified user statuses, whitelists, and compliance rules enforced by smart contracts.

Audit Trail

Generate immutable, on-chain records of all KYC checks and compliance decisions for easy reporting to auditors and regulators.

Quick Start

Installation

Install the SDK package via npm or yarn:

npm install @gitdigital-products/solana-kyc-sdk
# or
yarn add @gitdigital-products/solana-kyc-sdk

Basic Usage

Initialize the SDK and perform a basic check:

import { KYCClient } from '@gitdigital-products/solana-kyc-sdk';
import { Connection, clusterApiUrl } from '@solana/web3.js';

const connection = new Connection(clusterApiUrl('devnet'));
const client = new KYCClient(connection, YOUR_PROVIDER_CONFIG);

// Example: Check a user's wallet status
const walletAddress = '...';
const status = await client.getVerificationStatus(walletAddress);
console.log(`KYC Status: ${status}`);
