## How to Get Listed

Contributors are recognized when they:

- Submit code (merged PR)
- Contribute documentation
- Help in community forums
- Report bugs
- Translate content
- Organize events
- Provide funding
- Serve on governance bodies

Want to be a contributor? See [CONTRIBUTING.md](../CONTRIBUTING.md) to get started.

## Contributor Recognition Program

We believe in recognizing contributions in multiple ways:

- **Shoutouts** in community calls
- **Mentions** in release notes
- **Contributor badges** on Discord
- **Swag** for significant contributions
- **Grant eligibility** for active contributors
- **Maintainer nominations** for exceptional contributors
